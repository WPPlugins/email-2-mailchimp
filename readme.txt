Подтверждение подписки не требуется.Плагин разработал [Арсен Ибрагимов (OnArs)](http://ibragimov.me/ "Арсен Ибрагимов").== Installation ==Внимание! Плагин платный. Для работы плагина нужно приобрести уникальный ключ и привязать его к домену на котором используется плагин. Внимательно читайте нижеприведённую инструкцию.1. Загрузите папку плагина 'email-2-mailchimp' в '/wp-content/plugins/' на своём сервере.2. Активируйте плагин в разделе 'Плагины' в меню WordPress'a.3. Перейдите в раздел 'Параметры > eMail 2 MailChimp', введите свой MailChimp API ключ и выберите лист в который будут подписываться email'ы Ваших комментаторов.4. Для работы плагина необходимо приобрести уникальный ключ по адресу: http://ibragimov.me/magazin/email-2-mailchimp/ 5. Для активации ключа и привязки его к домену на котором вы используете плагин перейдите по адресу: http://e2m.ibragimov.me/ 6. Введите ключ и нажмите на кнопку "Проверить ключ". Если ключ верный, то появятся два поля для ввода домена и email адреса.7. Введите домен в формате "ibragimov.me" (без пробелов, без ковычек, без http:// и прочих специальных символов) и обязательно укажите контактный email адрес.8. Готово! Плагин установлен, активирован и готов к работе!В случае проблем пишите: barsnur@gmail.com== Changelog === 1.0.9 =* Исправлены мелкие баги= 1.0.0 =* Релиз!
